# diffdirUi


## 開発者向け情報 - for Developer

- 動作環境

node 0.12.x  ※node-sassで0.12を要求
npm 2.x  
electron 0.x ※1.x系では動きません。

- 実行

```
$ npm install
// electron ０.x系をバージョン指定でダウンロード
$ npm install -g electron-prebuilt@0
// node-php-binやphpスクリプトunpackedにコピー
$ gulp
$ electron .
```

- ビルド

```
$ npm run build
```
